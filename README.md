# Gamificação Sustentável SoulUp 🌱
Front-end desenvolvido para o Challenge FIAP utilizando HTML, CSS e JavaScript com foco em sustentabilidade e gamificação.


# Sobre o Projeto 📌

Desenvolvemos uma plataforma web criada para incentivar práticas sustentáveis através de desafios, ranking de usuários e acompanhamento de desempenho.

O objetivo do projeto é unir tecnologia, sustentabilidade e gamificação para criar uma experiência interativa e educativa.

# Funcionalidades 🚀

✅ Página inicial interativa

✅ Dashboard com informações dos desafios

✅ Sistema de ranking

✅ Página FAQ

✅ Página de contato

✅ Página sobre o projeto

✅ Página de integrantes

✅ Layout responsivo

✅ Interações com JavaScript

# Tecnologias Utilizadas 🛠️
- HTML5
- CSS3
- JavaScript
- Git & GitHub


# Objetivo Acadêmico 🎯

Este projeto foi desenvolvido como um Challenge do curso de Análise e Desenvolvimento de Sistemas da FIAP, com o objetivo de aplicar na prática os conhecimentos adquiridos ao longo do semestre nas disciplinas de Front-End Design Engineering.
